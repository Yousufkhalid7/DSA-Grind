28. Find the Index of the First Occurrence in a String
Approach: Traversal

Solution:
class Solution(object):
    def strStr(self, haystack, needle):
        i = 0
        m = len(needle)
        n = len(haystack)
        if needle == "":
            return 0
        while i <= n-m:
            if haystack[i:i+m] == needle:
                return i
            i+=1
        return -1
