268.Missing Number
Approach: Sorting

Solution:
class Solution(object):
    def missingNumber(self, nums):
        nums.sort()
        if nums[0] != 0:
            return 0
        
        for i in range( 1, len(nums)):
            if nums[i] - nums [i-1] == 2:
                return i
        return len(nums)
