22. Majority Element 2
Approach: Sort + Count ( Better Space Complexity but O(nlogn) due to sorting 

Solution:
class Solution(object):
    def majorityElement(self, nums):
        nums.sort()
        i = 1
        count = 1
        check = len(nums)//3
        result = []
        for i in range(1, len(nums)):
            if nums[i] == nums[i-1]:
                count += 1
            else:
                if count > check:
                    result.append(nums[i-1])
                count = 1
        if count > check:
            result.append(nums[-1])
        return result
Approach: Hash Frequency Count ( BETTER )
Solution:
class Solution(object):
    def majorityElement(self, nums):
        hmap = {}
        for num in nums:
            if num in hmap:
                hmap[num] += 1
            else:
                hmap[num] = 1
        result = []
        check = len(nums)//3
        for key in hmap:
            if hmap[key] > check:
                result.append(key)
        return result
