34. Find First and Last Position of Element in Sorted Array
Approach: Two Pointer (Wrong Approach)

Solution:
class Solution(object):
    def searchRange(self, nums, target):
        i = 0
        j = len(nums) - 1
        first, last = -1,-1
        while i <= j:
            if first == -1:
                if nums[i] == target:
                    first = i
                else:
                    i+=1
            if last == -1:
                if nums[j] == target:
                    last = j
                else:
                    j-=1
            if first != -1 and last != -1:
                break
        return [first,last]
