167. Two Sum II - Input Array Is Sorted
Approach: Two Pointers

Solution:
class Solution(object):
    def twoSum(self, numbers, target):
        left = 0
        right = len(numbers)-1
        while left<right:
            summ = numbers[right] + numbers[left]
            if summ == target:
                return left+1,right+1
            elif summ <= target:
                left += 1
            elif summ > target:
                right -= 1
        return [1,2]
