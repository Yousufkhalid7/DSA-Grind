80. Remove Duplicates from Sorted Array II
Approach: Two Pointers

Solution:
class Solution(object):
    def removeDuplicates(self, nums):
        if len(nums) <= 2:
            return len(nums)
        k=2
        for i in range(2, len(nums)):
                if nums[k-2] != nums[i]:
                    nums[k] = nums [i]
                    k+=1
        return k
