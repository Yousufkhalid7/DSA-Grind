238. Product of Array Except Self
Approach: Calculate the product(without self) by multiplying prefix and postfix elements

Solution:
class Solution(object):
    def productExceptSelf(self, nums):
        arr = [1]*len(nums)
        left = 1
        for i in range(len(nums)):
            arr[i] = left
            left = left * nums[i]

        right = 1
        for j in range(len(nums)-1, -1, -1):
            arr[j] =  arr[j]*right
            right = right * nums[j]

        return arr
