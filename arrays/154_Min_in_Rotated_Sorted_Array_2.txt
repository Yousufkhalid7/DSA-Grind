154. Find Minimum in Rotated Sorted Array ||
Approach: Classic Binary Search always finds the minimum element

Solution:
class Solution(object):
    def findMin(self, nums):
        left = 0
        right = len(nums) - 1
        while left < right:
            mid = (left+right)//2
            if nums[mid] > nums[right]:
                left = mid + 1
            elif nums[mid] < nums[left]:
                right = mid
            else:
                right -= 1
        return nums[left]
