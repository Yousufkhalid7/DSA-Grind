69. Sqrt(x)
Approach: Check for every integer

Solution:
class Solution(object):
    def mySqrt(self, x):
        if x == 0:
            return 0
        i = 1
        while i <= x:
            if i*i == x:
                return i
            elif i*i > x:
                return i-1
            else:
                i+=1
