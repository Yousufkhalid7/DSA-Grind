53. Maximum Subarray
Approach: Sliding Window

Solution:
class Solution(object):
    def maxSubArray(self, nums):
        current_sum = 0
        max_sum = nums[0]
        for n in nums:
            if current_sum < 0:
                current_sum = 0
            current_sum += n
            max_sum = max(current_sum, max_sum)
        return max_sum
