106. Construct Binary Tree from Inorder and Postorder
Approach: Brute Force O(n2) -> Last element of postorder - root node
We keep popping from postorder until that value in inorder has no value on its right, then we put left

Solution:
class Solution(object):
    def buildTree(self, inorder, postorder):
        if not inorder or not postorder:
            return None
        root_val = postorder.pop()
        root = TreeNode(root_val)
        mid = inorder.index(root_val)
        root.right = self.buildTree(inorder[mid+1:], postorder)
        root.left = self.buildTree(inorder[:mid], postorder)
        return root

Approach: Optimized O(n)
