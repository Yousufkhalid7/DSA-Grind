134. Gas Station
Approach: Greedy Method ( Calculate the difference ( gas - cost ) and if this ever goes below 0, then its the false index )

Solution:
class Solution(object):
    def canCompleteCircuit(self, gas, cost):
        if sum(gas) < sum(cost):
            return -1
        index = 0
        tank = 0
        for i in range(len(gas)):
            tank += (gas[i] - cost[i])
            if tank < 0 :
                index = i + 1
                tank = 0
        return index
